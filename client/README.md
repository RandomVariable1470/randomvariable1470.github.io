# Portfolio Client

The frontend for the portfolio website.

## Tech Stack
- React
- Typescript
- TailwindCSS
- Framer Motion
- React Three Fiber
- Shadcn UI

## Development

This client is intended to be run as part of the full-stack application from the root directory (`npm run dev`).

To run standalone:
```bash
npm install
npm run dev
```
Note: You will need the backend running for API features (contact form, GitHub data) to work.
